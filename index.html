Model: the green line reinvests each period's interest. In Per year mode the rate is nominal and split across periods — A = P(1 + r/n)^(nt), or A = Pe^(rt) continuous — so frequency only nudges the effective yield (APY). In Per period mode the full rate lands every step — A = P(1 + r)^(nt) — so frequency dominates: “3% per trading day × 252” is 1.03^252. That mode is a sensitivity tool, not a forecast — sustaining a per-period edge like that indefinitely isn't realistic. The amber line accrues the same rate but withdraws it, staying linear. Figures are nominal (not inflation-adjusted) and assume the rate holds every period.
